# fQGAM 0.1.0

* Initial release of the fQGAM package
* Core functions for functional quantile regression:
  - `prep_fqgam_data()`: Data preparation helper
  - `boot_pred_block()`: Block bootstrap inference
  - `boot_pred_wild()`: Wild bootstrap inference
  - `predict_fqgam()`: Prediction at reference curves
  - `boot_ci()`: Bootstrap confidence intervals
* Vignette demonstrating usage with DTI data
